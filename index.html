¿Qué es un Decorador?

Una función que "envuelve" a otra función para añadirle una nueva funcionalidad sin modificar su código interno.

2 / 13

Prerrequisito: Funciones como Objetos

En Python, las funciones pueden ser pasadas como argumentos a otras funciones.

def saludar():
    return "Hola"

def ejecutar_funcion(func):
    print(func())

ejecutar_funcion(saludar) # Imprime "Hola"

La Estructura de un Decorador

Un decorador es una función que define una función "envoltura" (wrapper) dentro de sí y la devuelve.

def mi_decorador(funcion_original):
    def envoltura():
        # Código extra ANTES de la función original
        print("Algo sucede antes.")
        
        funcion_original() # Se ejecuta la función original
        
        # Código extra DESPUÉS
        print("Algo sucede después.")
        
    return envoltura

Aplicar un Decorador (La Forma Larga)

Se pasa la función original al decorador y se reasigna el resultado.

def di_hola():
    print("¡Hola!")

# Aplicando el decorador manualmente
di_hola_decorado = mi_decorador(di_hola)
di_hola_decorado()

La Magia: Sintaxis @

Python nos da una forma mucho más limpia y legible de hacer lo mismo con el símbolo `@`.

@mi_decorador
def di_hola():
    print("¡Hola!")

di_hola() # Llama directamente a la versión decorada

Decoradores con Argumentos

Para que funcione con cualquier función, la envoltura debe aceptar *args y **kwargs.

def decorador_universal(func):
    def envoltura(*args, **kwargs):
        print("Antes de la llamada.")
        func(*args, **kwargs) # Pasamos los argumentos
        print("Después de la llamada.")
    return envoltura
8 / 13

Ejemplo con Argumentos

Ahora el decorador funciona con una función que recibe parámetros.

@decorador_universal
def saludar(nombre, saludo="Hola"):
    print(f"{saludo}, {nombre}")

saludar("Ana")

Manejar Valores de Retorno

La envoltura debe capturar y devolver el resultado de la función original.

def decorador_con_retorno(func):
    def envoltura(*args, **kwargs):
        print("Calculando...")
        resultado = func(*args, **kwargs)
        print("¡Cálculo finalizado!")
        return resultado # Muy importante
    return envoltura
10 / 13

Ejemplo con Retorno

Ahora podemos usar el valor que devuelve la función decorada.

@decorador_con_retorno
def sumar(a, b):
    return a + b

resultado_suma = sumar(5, 3)
print(f"El resultado es: {resultado_suma}") # 8

Caso de Uso: Medir Tiempo

Un decorador es perfecto para medir cuánto tarda una función en ejecutarse.

import time
def timer(func):
    def envoltura(*args, **kwargs):
        inicio = time.time()
        resultado = func(*args, **kwargs)
        fin = time.time()
        print(f"{func.__name__} tardó {fin-inicio:.4f}s")
        return resultado
    return envoltura

Conclusión

Los decoradores permiten extender la funcionalidad de funciones existentes de forma limpia, reutilizable y sin modificar su código.
